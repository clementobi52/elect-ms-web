🗳️ Election Monitoring System
Project Overview
This election monitoring application is a dual-platform solution designed to empower polling agents and situation room analysts. It ensures election integrity through real-time location validation, incident reporting, and secure result submission.

🏗️ Technical Architecture
The project utilizes a unified codebase approach to handle both field operations and administrative oversight:

Mobile App: Built with Expo / React Native for seamless cross-platform deployment.

Web Dashboard: Built with Next.js for high-performance admin and situation room monitoring.

Styling: Powered by Tailwind CSS for a consistent design language across web and mobile.

✨ Key Features
📍 Precision Tracking: Real-time geolocation tracking with polling unit validation to ensure agents are on-site.

📶 Offline First: Robust offline data storage and background synchronization for areas with poor connectivity.

📸 Secure Submissions: Direct upload of election results with support for high-quality images and PDF documentation.

🚨 Incident Reporting: Real-time fraud alert and incident reporting system for immediate situation room awareness.

📊 Live Monitoring: Situation room dashboard featuring live location tracking and distance metrics.

📱 Modern UX: User-friendly mobile interface with responsive bottom-tab navigation.
